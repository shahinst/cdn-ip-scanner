Fonts used by CDN IP Scanner V1.6
====================================

- Vazirmatn-Regular.ttf, Vazirmatn-Bold.ttf  → Persian text
- fa-solid-900.ttf   → Font Awesome 6 Free Solid (colored icons in app)
- fa-brands-400.ttf  → Font Awesome 6 Brands (GitHub, YouTube icons)

Font Awesome 6: https://fontawesome.com/
If fa-solid-900.ttf or fa-brands-400.ttf are missing, the app falls back to emoji.
