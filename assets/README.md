# Assets — استایل و تم برنامه

این پوشه تنظیمات ظاهری برنامه را از فایل اصلی جدا می‌کند.

## theme.json

- **themes**: رنگ‌های تم تیره و روشن (فقط چهار رنگ: سفید، خاکستری، نارنجی، مشکی).
- **layout**: اندازهٔ دکمه‌ها (یکسان برای همه)، فاصلهٔ کارت‌ها و سکشن‌ها، ضخامت حاشیه.
- **rtl**: تنظیمات راست‌چین برای زبان فارسی.

با تغییر `theme.json` می‌توانید بدون دست زدن به کد اصلی، ظاهر برنامه را عوض کنید.
